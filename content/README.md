# Content Production Folder

This is where all blog content lives for sdubmedia.com.

## How It Works
1. **Before writing anything** — read everything in `context/`. That's the voice, style, and examples.
2. **Drafts go in** `drafts/` — agent writes here, Geoff reviews before anything goes live.
3. **Approved content goes in** `published/` — once reviewed and approved.
4. **Images go in** `images/generated/` for AI-generated or `images/assets/` for real photos.

## Folder Map
- `context/` — brand voice, style guide, writing examples. Read this first, every time.
- `drafts/` — work in progress. Not live.
- `images/generated/` — DALL-E or fal.ai images for blog posts
- `images/screenshots/` — screenshots of the site or other references
- `images/assets/` — real SDub Media photos to use in posts
- `published/` — approved posts, ready to go live
