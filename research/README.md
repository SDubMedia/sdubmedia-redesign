# Research Folder — SDub Media SEO

This folder contains all SEMrush keyword data, competitor analysis, and topical maps
for sdubmedia.com. Pulled during a 7-day SEMrush trial in May 2026.

## How the Blog Agent Uses This

Before picking a topic, the agent reads:
1. `keywords/target-keywords.md` — picks the next keyword that hasn't been written about
2. `topical-map/supporting-posts.md` — checks which cluster fits
3. `competitors/content-gaps.md` — avoids overlap with competitor content that already dominates

## Folder Map

- `keywords/` — master keyword list, sorted by priority
- `competitors/` — tonefilms + others, content gaps
- `topical-map/` — the full topical authority structure
- `local-seo/` — Nashville suburb and local-intent keywords
- `audit/` — technical SEO issues on current sdubmedia.com
